# Auto Driving Car
Sensor Programming 2022-2 Team Project
<br><br>

## Tech Stack
- Raspberry Pi 4
- python3
- OpenCV3
- YOLOv3
<br><br>

## Description
Our Auto Driving Car Project includes the following features.
- line detection
- traffic light detection
- speed sign detection
- obstacle detection
